# my-rime-config

我的个人 RIME 输入法配置，基于 [雾凇拼音 (rime-ice)](https://github.com/iDvel/rime-ice)。

## 使用平台

- **Windows**: 小狼毫 (Weasel)
- **macOS**: 鼠须管 (Squirrel)
- **Linux**: ibus-rime / fcitx5-rime
- **iOS**: 仓输入法 (Hamster)
- **Android**: 同文输入法 (Trime)

## 个人定制

| 文件 | 说明 |
|------|------|
| `default.custom.yaml` | 启用方案：雾凇拼音 + Melt Eng |
| `weasel.custom.yaml` | Windows 小狼毫配色：ink（暗色） |
| `custom_phrase.txt` | 自定义短语（常用单字、词汇、英文等） |

## 输入方案

- 雾凇拼音 (rime_ice) — 全拼主方案
- Melt Eng (melt_eng) — 英文混输

## 安装

```bash
git clone https://github.com/alonelypigeon/my-rime-config.git
```

将文件复制到 RIME 配置目录后重新部署即可。
